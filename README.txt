
Legaltech LT - Demo funcional para feria
--------------------------------------

Contenido del paquete:
- index.html   -> página principal (demo funcional)
- assets/logo.png -> logo de la demo
- /assets      -> carpeta de recursos

Instrucciones rápidas para desplegar en Netlify (arrastrar y soltar):
1. Descomprime este ZIP en tu computador (opcional).
2. Ve a https://app.netlify.com/ y crea una cuenta (puedes usar Google).
3. En el panel: Add new site -> Deploy manually -> Drag & drop your site folder or the contents.
4. Netlify publicará el sitio y te dará un enlace tipo https://your-site-name.netlify.app
5. Si el sitio no carga bien, revisa que el archivo index.html esté en la raíz del paquete.

Nota: Esta versión es una demo funcional. El "pago" es simulado y el PDF se genera localmente en el navegador.
